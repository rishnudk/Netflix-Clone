## ğŸ“¦ Installation

1. Clone this repository:
  
   git clone https://github.com/rishnudk/netflix-clone.git
  

2. Navigate to the project folder:
  
   cd netflix-clone
  

3. Install dependencies:
    
   npm install

   Copy your Firebase config and replace it inside `firebase.js`.
